# Research priorities for global food security under extreme events

Data and script for reproducing the final results shown in Mehrabi et al., Research priorities for global food security under extreme events, One Earth (2022), https://doi.org/10.1016/j.oneear.2022.06.008.

Simply download and read the Mehrabi2022_EEGFS.pdf or the Mehrabi2022_EEGFS.Rmd file from which it was created.
